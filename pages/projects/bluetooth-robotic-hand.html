Bluetooth Robotic Hand

An Arduino-powered robotic hand built with servos and flex sensors to help users with limited hand mobility clutch and hold objects more easily.

Robotics / Accessibility
View GitHub
Project Image

Overview

This project explores assistive robotics by translating hand or sensor input into controlled robotic finger movement for object gripping.

Technical Focus

The system centers on Arduino control, servo actuation, flex sensor readings, and the mechanical response needed for a reliable clutching motion.

Build Notes

  • Arduino-based control logic
  • Servo-driven finger movement
  • Flex sensor input for motion control
  • Accessibility-focused design goal
