Productivity AI

A Python productivity analysis tool that classifies running processes and turns activity data into a visual breakdown of productive, unproductive, and unknown time.

First AI Venture
View GitHub

Overview

Productivity AI was an early 2021/2022 project built to scan active computer processes and help users understand how their open applications affected their focus.

Technical Focus

The tool let users define which processes counted as productive or unproductive, tracked how long those programs stayed open, and summarized the results with Python data processing and visual charts.

Build Notes

  • Scanned the computer for running processes during a tracking session
  • Compared process names against user-selected productivity categories
  • Calculated time totals for productive, unproductive, and unknown activity

Goal

The goal was to further implement AI/ML by generating productivity suggestions and reminders from time to time, especially when high unproductive activity was measured.
